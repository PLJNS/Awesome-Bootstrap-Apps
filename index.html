Awesome Apps

I always had this little craving to mock-up an app spring board with some standard apps, just for some fun.

These use Font Awesome and CSS to simulate app icons, and use Apple's various design sizes

I used Bootstrap and Font Awesome to make the actual icons, and BrandColors for the, well, brand colors.

Big thank you to MaxCDN and their Bootstrap CDN.

How to

  1. Download, copy-paste, clone apps.css
  2. Add it to your HTML header.
  3. To make a Facebook icon appear, type <div class="icon facebook-icon iphone-retina"><p><i class="icon-facebook"></i></p></div>.
    • If you want LinkedIn, replace all instances of "facebook" with "linkedin".
  4. Enjoy!
